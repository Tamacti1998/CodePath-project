# 🚐 ShuttleFlow

ShuttleFlow is a transportation platform designed to help college students move around campus and nearby areas easily—similar to Uber, but tailored for student communities. It connects riders with available drivers in a simple and efficient way.

---

## 📌 Features

- Request a ride in real-time
- View available drivers
- Basic ride matching system
- User-friendly interface for students
- Scalable concept for campus transportation
